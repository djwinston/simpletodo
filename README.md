# simpletodo

## Project setup

```php
npm install
```

### Run NodeJS server

```php
npm run server
```

### Compiles and hot-reloads for development

```php
npm run serve
```

### Compiles and minifies for production

```php
npm run build
```

### Run your unit tests

```php
npm run test:unit
```

### Lints and fixes files

```php
npm run lint
```
